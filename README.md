Fast Food Restaurant System (COAL Project)

A console-based **Fast Food Restaurant System** developed using **x86 Assembly Language (MASM)** with the **Irvine32 library**.  
This project simulates a real fast-food ordering system where users can select items, enter quantities, and generate a final bill.
